In order to work with the scrapy data:

  * Copy the folder at your desired destination
  * Move to bookBoarn folder, and open the terminal there
  * Run 'scrapy' command and list the current working project
  * To list the scrapy spiders, run 'scrapy list'; which lists:
	book16
	book17
	book18
	book19
	book20
	book21
  * To run a spider, run the command 'scrapy crawl [spidername]
	eg. scrapy crawl book20
